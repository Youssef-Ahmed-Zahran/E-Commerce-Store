# 🛍️ Full-Stack E-Commerce Platform

A complete e-commerce solution built with modern web technologies, featuring comprehensive product management, secure payments, and an advanced admin dashboard.

## 🎯 Core Features

### 🔐 User Management & Authentication

* ✅ Secure JWT-based authentication with HTTP-only cookies
* ✅ User registration, login, and profile management
* ✅ Role-based access control (Admin & Customer roles)
* ✅ Password encryption with bcrypt

### 📦 Product Management

* 📦 Full CRUD operations for products
* 🖼️ Image upload functionality with Multer
* 🏷️ Category-based product organization
* ⭐ Product ratings and reviews system
* 🔍 Advanced search and filtering (by category, price range, brand)
* 📄 Pagination for better performance
* 🔥 Featured products carousel (Top & New arrivals)

### 🛒 Shopping Experience

* 🛒 Fully functional shopping cart with real-time updates
* ❤️ Wishlist/Favorites functionality
* 🎨 Responsive UI with Tailwind CSS
* 🌙 Modern gradient-based design theme
* 📱 Mobile-first responsive design

### 📋 Order Management

* 📋 Complete order lifecycle (Create, View, Track)
* 💳 PayPal payment integration
* 📦 Order status tracking (Paid/Unpaid, Delivered/Pending)
* 🚚 Shipping address management
* 📊 Order history with pagination

### 📊 Admin Dashboard

* 📊 Sales analytics with interactive charts (ApexCharts)
* 👥 User management (View, Edit, Delete users)
* 📦 Product inventory management
* 🏷️ Category management
* 📈 Real-time statistics (Total Sales, Orders, Customers)
* 📅 Sales trends by date
* 📋 Order management and delivery status updates

## 🛠️ Tech Stack

### ⚡ State Management & Data Fetching

* Redux Toolkit for global state management
* RTK Query for efficient API calls and caching

### 🔒 Security Features

* HTTP-only cookies for token storage
* Password hashing with bcrypt
* Protected routes (Frontend & Backend)
* Admin authorization middleware
* CORS configuration
* Input validation

### 📈 Performance Optimizations

* Pagination on all major data lists
* Image optimization
* Lazy loading
* RTK Query caching strategies
* Efficient database queries
